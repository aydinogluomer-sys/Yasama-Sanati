# Anti-Patterns

## Do Not

- Add gradients everywhere.
- Add glow for the sake of visual energy.
- Stack too many animations.
- Use generic wellness card templates.
- Copy SaaS conventions blindly.
- Let decorative layers obscure conversion content.
- Invent new colors and spacing without a system reason.
- Ignore accessibility in the name of polish.

## Watch For

- Template-like footer structure
- Overpacked hero copy
- Repetitive section messaging
- Decorative motion that steals attention
- Desktop layouts that break on mobile
