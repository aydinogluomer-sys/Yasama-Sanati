# Spacing

## Rhythm

- Small gap: 8–12px
- Medium gap: 16–24px
- Large gap: 32–48px
- Section gap: 96–160px

## Layout Rules

- Use large negative space to create editorial calm.
- Prefer fewer stacked elements with stronger separation.
- Keep mobile spacing generous enough to avoid “shrunk desktop” feeling.
- Keep footer and hero spacing more deliberate than utility-heavy sections.

## Content Width

- Body text: narrow to medium measure
- Section content: balanced, airy, and readable
- Full-width content only when the visual logic justifies it

## Rules

- Avoid dense vertical packing.
- Avoid random spacing jumps between adjacent sections.
- Keep rhythm consistent across the page.
