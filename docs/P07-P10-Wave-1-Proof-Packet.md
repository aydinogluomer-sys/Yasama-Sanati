## 1. Wave 1 Visual Brief Summary

This is **R2 Phase 07 — Wave 1 Visual Brief only**.

## 2. Creative Objective

The opening should feel like entering a calm, intelligent, editorial academy of embodied life practice — not a spa, not a luxury retreat, not a generic wellness landing page.

The user should experience:

```text
Karşılanma:
"I have arrived somewhere calm, serious, human and considered."

Merak:
"There is more depth here. I want to understand the method, the atmosphere and the promise."
```

Hero, the bridge, and Introduction must work as one continuous opening gesture. The Hero welcomes; the bridge slows attention; the Introduction begins inquiry. Together they form the first signature moment:

```text
Opening Breath
```

Opening Breath is not a spectacle. It is the feeling of the page inhaling, holding attention, then guiding the visitor into curiosity.

## 3. Opening Breath Definition

Opening Breath is the first authored moment of the site.

It should feel like:

```text
arrival
attention
slow confidence
editorial pause
soft invitation
controlled curiosity
```

It should not feel like:

```text
hero animation
scroll trick
Awwwards clone
wellness template
luxury resort opening
cinematic trailer
3D/tech showcase
```

The transition from Hero into Introduction should feel almost ritualized, but not mystical. The user should not feel "an effect happened." The user should feel "the site guided my attention."

Current problem hypothesis:

```text
- The opening may still risk generic wellness language.
- Empty space may not yet feel fully authored.
- Typography may not yet carry enough editorial authority.
- Hero-to-Introduction may not yet feel like a designed ritual.
- Mobile may risk becoming a squeezed version of desktop.
```

This brief does not prescribe fixes. It defines what the later visual proof must solve.

## 4. Hero Brief

### Desired Hero Character

The Hero should establish the world of Yaşama Sanatı immediately:

```text
academic but warm
calm but not sleepy
premium but not luxury-resort
cinematic but not trailer-like
human but not stock wellness
intelligent but not mystical-tech
editorial but not fashion-portfolio
```

### Hierarchy

The Hero hierarchy should feel authored and intentional. The first screen must answer:

```text
Where am I?
What kind of institution is this?
Why does it feel different from generic wellness?
Why should I keep scrolling?
```

The main title should carry intellectual and emotional weight. Supporting text should not over-explain. It should invite entry.

The Hero should avoid the common landing-page rhythm of:

```text
big headline
subtext
button
pretty image
```

Instead, the proof should explore a more editorial opening structure where type, image, spacing and silence feel composed.

### Typography Behavior

Typography should feel like a serious editorial publication, not a wellness app.

Desired qualities:

```text
measured scale
controlled line length
strong typographic pause
clear hierarchy
subtle academic confidence
breathing room around key phrases
```

Avoid:

```text
oversized inspirational slogan
generic serif luxury look
thin unreadable display type
template-like centered hero stack
motivational quote energy
```

### Image / Crop Logic

Any image direction should feel observed, not advertised.

Preferred image behavior:

```text
partial crop
human presence without stock-photo directness
texture / light / body / space rather than literal wellness pose
quiet asymmetry
editorial framing
```

Avoid:

```text
smiling stock wellness person
yoga/spa cliché
resort interior
AI-perfect human face
overly cinematic retreat shot
mystical light portal
```

### Whitespace Role

Whitespace must feel authored. It should create calm, hierarchy and attention — not emptiness.

The desktop proof should show whether whitespace is:

```text
holding the composition
guiding the eye
creating editorial confidence
making the opening slower and more premium
```

If the space feels accidental, the proof fails.

### Opening Rhythm

The first screen should not rush. The page should feel like it has an internal tempo:

```text
first glance: calm authority
second glance: human atmosphere
third glance: curiosity
scroll invitation: soft movement into Introduction
```

### Restrained Motion Intent

Motion should be imagined as minimal and breath-like. Motion is not the main idea; attention is.

Allowed motion character:

```text
slow reveal
soft settling
subtle parallax-like depth only if necessary
breath-paced transition
calm scroll response
```

Rejected motion character:

```text
flashy reveal
scrub-heavy performance
GSAP showcase
3D reveal
particles
blobs
glassmorphism
cinematic trailer move
```

### Desktop Intent

Desktop should feel spacious, editorial and composed. It should use scale and whitespace confidently, but not become empty.

Desktop proof must make the opening feel like a designed publication spread translated into web.

### Mobile Intent

Mobile must be independently composed.

It should not compress the desktop layout. It should re-author the opening for one-hand reading, small-screen hierarchy and slower vertical rhythm.

Mobile must preserve:

```text
calm
legibility
editorial authority
Opening Breath
```

## 5. Hero-to-Introduction Bridge Brief

The bridge is the core of Opening Breath.

It should not be treated as a decorative transition. It should be the moment where the user's attention shifts from arrival into inquiry.

Desired bridge feeling:

```text
the page exhales
the Hero releases
the Introduction appears as a continuation, not a new block
curiosity begins without noise
```

The bridge should prepare **Merak** by changing the user's attention state.

It should not:

```text
show off scroll technology
feel like a GSAP demo
introduce 3D
use random cinematic camera movement
create a hard section break
overpower the Hero or Introduction
```

The correct bridge is subtle enough that the user may not consciously notice it as an effect, but strong enough that the opening feels authored.

The bridge should feel like:

```text
breath
attention
arrival
soft invitation
```

not:

```text
animation
transition trick
portfolio effect
tech reveal
```

## 6. Introduction Brief

Introduction should transform welcome into curiosity.

It should answer:

```text

What intellectual/emotional promise is being made?
```

Introduction should not feel like a normal "about this method" text block. It should feel like the first page of an editorial essay.

### Editorial Pacing

Introduction should have a controlled reading rhythm.

Desired qualities:

```text

intentional pauses
strong line breaks
quiet emphasis
typographic contrast without noise
```

It should avoid:

```text
large uninterrupted copy block
generic wellness statement
centered inspirational paragraph
sales copy
overly poetic vagueness
```

### Text / Image Relationship

The text and image should not simply sit side by side. They should create tension, rhythm or inquiry.

Possible proof-level directions:

```text
text leads, image observes
image interrupts the reading rhythm softly
caption-like detail creates academic tone
crop creates curiosity rather than decoration
```

The Introduction image behavior should continue the Hero's world, not restart the visual language.

### Vertical Rhythm

The vertical spacing should feel like a slow editorial descent from Hero. The section should not appear as a separate block dropped under the Hero.

The rhythm should feel continuous:

```text
Hero = arrival
Bridge = breath
Introduction = first inquiry
```

### Curiosity-Building

Introduction should deepen the site without explaining everything.

It should create questions like:

```text
What is the method?
Who is this for?
How does this change daily life?
Why does this feel more serious than wellness?
```

### Desktop Intent

Desktop Introduction should feel like an editorial spread: composed, asymmetric enough to feel authored, but calm enough to remain readable.

### Mobile Intent

Mobile Introduction should be rewritten visually for vertical rhythm. It should not simply stack desktop elements.

Mobile proof must show:

```text

strong type hierarchy
no fragile line breaks
no cramped image/text relationship
no squeezed desktop residue
```

# Phase 08 — Desktop Visual Proof Direction

## 1. Phase 08 Status

* This is **Phase 08 only**.
* This document defines **desktop visual proof direction only**.
* **Mobile proof has not started** and remains blocked until desktop proof is creatively approved.
* **Implementation is not authorized**.
* No code, component tasking, build direction, or production prompt generation is included here.
* Phase 08 exists only to turn the accepted Phase 07 brief into a **clear desktop opening proof target** for:
  * Hero
  * Hero-to-Introduction bridge
  * Introduction

## 2. Desktop Opening Concept

### Core visual idea

The desktop opening should feel like a **measured inhale**: not a spectacle, not a poster, not a lifestyle brochure, but an authored editorial welcome that gives the visitor the sense that they are entering a serious, calm, deeply considered academy.

### Opening Breath

"Opening Breath" is the first signature moment. It should express:

* **arrival before explanation**
* **presence before persuasion**
* **calm authority before information density**
* **curiosity through restraint, not through noise**

The opening is not trying to impress with complexity. It is trying to create **trusting attention**.

### Emotional movement

The sequence should move the visitor from:

* **Karşılanma**
  into
* **Merak**

That movement should happen through a controlled shift:

1. **Hero** establishes atmosphere, identity, and seriousness.
2. **Bridge** softens the handoff and turns passive looking into active attention.
3. **Introduction** introduces inquiry, method, and the first clear sense that there is a worldview here worth exploring.

The user should feel:

> "I am in a thoughtful place."
> then
> "There is something more precise here than a generic wellness experience."

## 3. Hero Desktop Composition

### Layout logic

The Hero should feel **authored rather than centered-by-default**.

The composition should be asymmetrical, calm, and deliberate:

* one dominant visual field
* one primary typographic anchor
* one secondary textual intelligence layer
* generous negative space that feels intentional, not vacant

The page should not read like:

* a retreat poster
* a luxury villa promo
* a slow cinematic stock reel

It should read like a **serious editorial opening scene**.

### Hierarchy

The hierarchy should be unmistakably clear:

1. **Primary:** Yaşama Sanatı brand statement
2. **Secondary:** short intellectual / emotional framing line
3. **Tertiary:** minimal orientation cues only

The Hero should not try to say too much. Its job is not explanation. Its job is **tone, authorship, and attention**.

### Typography behavior

Typography should behave like architecture, not decoration.

* The main display should feel calm, large, and grounded.
* It should carry editorial gravity without becoming fashion-editorial.
* Serif presence should feel intentional and dignified.
* Supporting text should be quieter, more restrained, and structurally supportive.
* Handwritten or human accent, if present at all, must remain extremely restrained and meaningful.

The Hero type should not feel:

* trendy
* mystical
* poetic in a vague way
* performatively premium

It should feel **confident, spacious, and precise**.

### Image / crop behavior

The image logic should avoid the "beautiful but generic wellness landscape" trap.

The crop should feel:

* authored
* close enough to have point of view
* atmospheric without becoming scenic wallpaper

The visual should not function as a postcard.
It should function as an **environmental field for the typography**.

Preferred feeling:

* quiet topography
* air
* depth
* natural atmosphere
* subtle human presence only if it strengthens embodiment rather than stock-lifestyle coding

### Whitespace role

Whitespace should function as **breathing structure**.

It should:

* intensify the calm
* give the typography authority
* allow the image to feel selected rather than overloaded
* prepare the bridge into the next section

Whitespace must not feel like unresolved emptiness.
It must feel like **editorial confidence**.

### First-screen rhythm

The first-screen rhythm should be:

* immediate atmosphere
* quick legibility
* pause
* invitation downward

The visitor should understand the tone quickly, then feel encouraged to continue — not because something flashy pulls them, but because the opening feels trustworthy and composed.

### Restrained motion intent

Motion, if implied, should feel like:

* atmospheric settling
* very slow reveal
* subtle emphasis shift
* breath-like continuity

Not:

* dramatic parallax demo
* masked spectacle
* showreel opening
* scroll trick

## 4. Hero-to-Introduction Desktop Bridge

### Transition feeling

The bridge should feel like a **continuation of attention**, not a separate effect.

It should behave like:

* exhale into inquiry
* soft change in density
* a gentle shift from aura to meaning

### Visual continuity

The bridge should connect Hero and Introduction through continuity in:

* tonal atmosphere
* typographic dignity
* spatial cadence
* image logic
* emotional restraint

The user should never feel:

> "Hero ended, now a new layout begins."

Instead they should feel:

> "The opening is unfolding."

### Karşılanma → Merak movement

Hero carries Karşılanma through atmosphere and calm recognition.
The bridge begins Merak by slightly increasing interpretive tension.

That means the bridge should:

* reduce pure scenic dominance
* retain emotional calm
* introduce a stronger sense of intellectual orientation
* guide the eye toward structured reading without becoming abrupt

### What it must not be

The bridge must not become:

* a GSAP showcase
* a dramatic transformation scene
* a 3D moment
* a typographic stunt
* a theatrical scroll event

It should feel **ritualized, not spectacular**.

### Desired effect

The visitor should feel:

* more focused
* slightly closer
* more ready to read
* more curious about the philosophy behind the academy

## 5. Introduction Desktop Composition

### Editorial pacing

The Introduction should feel like the first true act of reading.

Its pacing should be:

* slower than conventional marketing
* clearer than poetic abstraction
* more authored than a standard image-and-copy section

The section should not read as:

* "here is some body copy after the hero"
* a generic brand manifesto block
* a split-layout with decorative image support

It should feel like **an editorial inquiry panel**.

### Text / image relationship

The relationship between text and image should be **interdependent**.

The image should not feel isolated or dropped in.
The text should not feel like it occupies one side while the image passively occupies the other.

Instead:

* the image should deepen the emotional field of the text
* the text should frame the image intellectually
* any supporting information should act as quiet evidence, not as marketing badges

The relationship should create:

* trust
* seriousness
* curiosity

### Vertical rhythm

The Introduction should have a more disciplined vertical cadence than the current likely state.

It should create:

* less unresolved upper emptiness
* more deliberate entry into the text
* a stronger sequence from heading → framing thought → supporting detail

The visitor should feel guided, not dropped into a void.

### Curiosity-building

Curiosity in this section should come from:

* precision
* restraint
* selective revelation
* the sense that the academy has a coherent philosophy

It should not come from hype or mystery theatre.

Merak here should feel like:

> "I want to understand this approach."

### How it avoids a normal wellness text block

The Introduction must avoid:

* overlong centered rhetoric
* stock calm imagery
* vague inspirational tone
* symmetrical brochure logic
* oversized empty space with weak structure

It becomes distinct by:

* stronger editorial composition
* more purposeful image placement
* calmer but more exact typography
* a visible relationship between thought, body, and place

## 6. Desktop Visual Proof Frame List

### Frame 01 — Hero first impression

**Emotional purpose**
Establish Karşılanma through calm authority, atmosphere, and presence.

**Composition**
A spacious, authored first viewport with a dominant atmospheric image field, a clearly anchored title mass, and a restrained secondary text area. The composition should feel asymmetrical and intentional.

**Typography**
Large editorial display voice with strong dignity and restraint. Supporting type is compact, quiet, and intelligent. No over-poetic sprawl.

**Image / crop**
A non-generic atmospheric crop: suggestive, grounded, and selective. It should avoid postcard beauty and avoid polished resort cliché.

**Motion feeling, if any**
Stillness first, then the smallest sense of life — like air or attention settling.

**Rejection risks**

* looks like a retreat advertisement
* feels too scenic
* feels like a wellness poster
* relies on beauty rather than composition
* typography feels decorative instead of authored

### Frame 02 — Bridge / Opening Breath transition state

**Emotional purpose**
Carry the visitor from receiving the atmosphere into wanting to understand the world behind it.

**Composition**
A transitional state where Hero still lingers in tone, but the structure begins to orient the user toward reading. The composition should feel continuous, not cut into a new mode.

The primary display recedes in dominance. Supporting structure becomes slightly more legible and intentional. Text begins to behave more like editorial inquiry.

**Image / crop**
The visual field becomes less like an opening scene and more like a partner to thought. Cropping and image mass should support transition, not novelty.

**Motion feeling, if any**
A slow handoff, subtle and breath-like. No flourish.

**Rejection risks**

* transition reads as effect rather than continuity
* theatrical scroll moment
* abrupt tonal switch
* over-designed seam
* feels like a demo rather than a narrative passage

### Frame 03 — Introduction first inquiry

**Emotional purpose**
Establish Merak: the visitor senses there is a considered methodology and wants to continue.

**Composition**
An editorial arrangement where text, image, and supporting information feel intentionally related. The section should feel grounded, legible, and distinct from a generic split section.

**Typography**
Clear hierarchy between main thesis, supporting statement, and quieter evidence. Typographic scale should feel more academic/editorial than lifestyle-marketing.

**Image / crop**
Human and environmental imagery should feel embodied, real, and selective — not stock, not too perfect, not spiritually vague.

**Motion feeling, if any**
Minimal. The emphasis is on compositional clarity and calm pacing.

**Rejection risks**

* becomes a standard text block with image
* unresolved empty space
* image feels detached
* curiosity too weak
* mobile implications appear fragile even in desktop logic

## 3. Mobile Hero Composition

* one strong typographic anchor
* one short orientation layer
* enough negative space to breathe

Preferred mobile image behavior:

```text
vertical crop with point of view
partial atmosphere rather than full scene
texture / air / body / place
```

Avoid:

```text
hero-as-wallpaper
resort-like view
```

Preferred rhythm:

```text

supporting explanation
quiet continuation
```

The reader should not have to interpret layout logic. The structure should feel calm and inevitable.

### Text / image relationship

The image should not simply appear "between paragraphs."

It should either:

* deepen the emotional context of the text
* act as a visual pause
* make the concept feel embodied
* create curiosity through crop and restraint

The text should frame the image intellectually, not caption it like a brochure.

### Vertical rhythm

Mobile vertical rhythm is the primary design tool.

The Introduction should use:

* short but meaningful text groupings
* controlled spacing between thought units
* image placement as pause, not decoration
* clear hierarchy between thesis and support

It must not feel like a long uninterrupted essay, nor like a fragmented social-media layout.

Merak should come from precision and restraint.

The Introduction should make the user think:


Curiosity should not come from mystery, spectacle, or vague spirituality.
