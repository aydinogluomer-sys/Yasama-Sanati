## 1. Purpose and Authority

R3 exists to satisfy the R2 prerequisite: *"Wave 1 Visual Brief cannot start until R3 Creative Toolchain & Reference System is accepted or an equivalent toolchain decision is logged."* It defines the **references, proof tools, AI visual/video policy, IP/copyright safety rules, reference-audit standard, and proof-generation standards** that must be in place before any Wave 1 visual work begins.

R3 is a **governance document**. It:

- sets AI image/video guardrails,

## 3. Toolchain Decision Principles

1. **References inform, they do not author.** Approved references may shape rhythm, spacing, pacing, and restraint — never brand identity, composition, copy, code, or motion system by direct copy.
2. **Direction is fixed.** No reference reopens R1's Editorial Wellness Academy + Restrained Meridian Intelligence direction.
3. **Restraint over spectacle.** The default is the least amount of motion/effect that achieves the intent. "Awwwards feel" is never a reason to add a dependency.
4. **Proof before code.** Tools here generate *proof and understanding*, not production assets or implementation.
5. **License before use.** Nothing is copied from unknown- or incompatible-license sources.
6. **Protected areas are off-limits.** No reference or tool is used to reshape Innovation internals, SignatureTypeScene's role, Footer closure, or the core palette.
7. **Accessibility & reduced-motion are non-negotiable.** Any referenced motion must have a defined reduced-motion equivalent before it can influence implementation.
8. **Agent separation holds.** Agent 01 selects/approves references and generates proof; Agent 02 does risk review; Agent 03 plans/implements only authorized narrow scopes.

## 13. Reference Audit Standard

Before any source influences proof or implementation, it must pass this audit (owner: Agent 02 for risk; Agent 01 for creative fit):

1. **Classification check** — confirm the source's tier (§4 approved / §5 conditional / §6 quarantined / §7 forbidden-to-copy).
2. **License check** — identify license; unknown/incompatible → no code or asset use; inspiration-only at most, and quarantined by default.
3. **IP/identity check** — no recognizable brand/studio/creator identity, no living-creator style imitation, no franchise/person likeness (ties to §7, §10).
4. **Direction-fit check** — does the abstracted influence support Editorial Wellness Academy without drift? If it pulls toward luxury/mystical/SaaS/agency, reject.
5. **Protection check** — confirm it does not touch Innovation internals, SignatureTypeScene role, Footer closure, or core palette.
6. **Motion/a11y check** — any motion influence has a defined reduced-motion equivalent.
7. **Extraction record** — record *what* was taken (rhythm/pacing/anti-pattern) and *what was explicitly not copied* (identity/layout/code/copy/motion system).

**Proof-generation standard:** every proof artifact is (a) on-direction, (b) clearly labeled proof (not production), (c) badge-free (no dev/error/issue badges), (d) IP-safe per §10, (e) produced at both desktop and mobile where the phase requires it, and (f) accompanied by the extraction record above when derived from a reference.

- Any reference begins to override R1.

A stop condition is a hard halt: pause, correct, and re-audit before any further reference influence or proof work continues.
