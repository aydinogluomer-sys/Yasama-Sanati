# Conversion Strategy

## Primary Conversion

Contact / enquiry for sessions or training.

## Secondary Conversions

- Explore programs
- Learn about the academy
- Read the story
- Review the 3D meridian experience

## User Paths

### 1. Breath seeker

Needs reassurance, clarity, and a fast route to contact or the right program.

### 2. Energy-work explorer

Needs trust, method, and a visible connection between practices and outcomes.

### 3. Learner

Needs curriculum direction, credibility, and a simple path into programs.

### 4. Skeptical but curious visitor

Needs proof, calm language, and low-friction navigation.

## Current Page Flow

1. Hero
2. Introduction
3. Wellness / approach
4. Innovation
5. Story
6. Retreat
7. Form
8. Footer

## Flow Rules

- The hero must tell the user what the site offers within seconds.
- Introduction must remove ambiguity.
- Program and method sections must reduce uncertainty.
- Trust sections must answer “why this brand”.
- The form must feel like a guided next step, not a generic lead capture block.
- The footer should close the page with confidence and clarity.

## CTA Hierarchy

Primary:

- Contact / session request

Secondary:

- Explore programs
- Meet the instructors

Tertiary:

- Read the story
- Review the 3D experience

## Mobile Conversion Rules

- Primary CTA should remain visible early.
- Content must not rely on large horizontal layouts.
- Form inputs must stay readable and touch-friendly.
- Footer links should wrap cleanly without collapsing hierarchy.

## Trust Signals Needed

- Clear academy description
- Strong method explanation
- Instructor credibility
- Program structure
- Contact reliability
- Privacy / form reassurance

## Success Definition

The user should understand:

1. What Yaşama Sanatı is.
2. Why it is trustworthy.
3. What action to take next.
