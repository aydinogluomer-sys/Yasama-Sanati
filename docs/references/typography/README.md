# Typography References — Yaşama Sanatı (Phase 2)

Source: provided ChatGPT screenshots (`Elementis-SOTD/Screenshots/`). These define **behavior** to
translate into the brand world — **never copy literally** (Phase 2 §2.2 / §2.10).

Direction (fixed): **high-contrast serif + premium grotesk sans**. Font budget: Cormorant Garamond ·
Basis Grotesque Pro · Kisthe · Space Mono.

## 01 — Outline vs High-Contrast Serif (`01-outline-vs-high-contrast.jpg`)
Covers reference 01 (outline), 02 (high-contrast serif) and 06 (comparison).
- **Outline typography** — ACCEPTED as overlay / motion / atmospheric layer only. Ghost serif word,
  scroll-drawn, low opacity. Never primary readable content (no body/nav/buttons/forms).
- **High-contrast serif** — ACCEPTED as the main display direction → Cormorant Garamond (hero, section
  headlines, manifesto, footer close). Not for body/UI.
